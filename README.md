# Theta Tau PNM Study Guide

This study guide is updated weekly to help PNMs prepare for upcoming quizzes.

## Important Information
- Quizzes are **cumulative**
- Each quiz has a time limit of about **15–20 minutes**
- The Final quiz has a **longer** time limit
- You must score **80% or higher** to pass

## Purpose
This study guide is designed to closely match the material that PNMs are expected to know for their quizzes. Use it consistently to review, practice, and stay prepared each week.

## Features
- Weekly updated quiz material
- Mastery tracking
- Shuffle questions feature
- Progress and stats saved with local storage
- Improved refresh/reset behavior
- Redesigned interface and study flow
